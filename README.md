# Dawn-of-War-Definitive-Edition-Script
Small fan-made script for **Dawn of War - Definitive Edition** with handy key bindings for fog, resources, speed, and more.

## Installation

Place the `autoexec.lua` file in the main *Dawn of War – Definitive Edition* game folder.

## Key Binding Explanations

| Key | Function                  | Description                                                                                                                                    |
| --- | ------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------- |
| F1  | `toggle_fog()`            | Toggles the map’s visual fog on and off.                                                                                                                |
| F2  | `give_resources()`        | Instantly grants resources to the current player: <br>- Necrons: +100,000 Power <br>- All other races: +100,000 Requisition and +100,000 Power |
| F3  | `switch_player_control()` | Switches control between the local player and the AI.                                                                                          |
| F4  | `toggle_instant_build()`  | Enables or disables instant building for structures.                                                                                           |
| F5  | `setsimrate(5)`           | Slows the game for easier testing or cinematic purposes.                                                                                       |
| F6  | `setsimrate(8)`           | Restores the game speed to its normal rate.                                                                                                    |
| F7  | `setsimrate(40)`          | Increases the game speed for faster matches or testing.                                                                                        |
| F8  | `toggle_fow()`            | Reveals or hides the entire map.                                                                                                               |
| F9  | `toggle_screenmode()`     | Hides or shows the game’s interface, useful for screenshots.                                                                                   |

## Notes

This script works with other mods.  
This script also works with the original *Dawn of War* and its expansions.
